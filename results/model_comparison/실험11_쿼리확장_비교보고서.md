## 쿼리 확장 방식

```
[기존 방식]
사용자 질문 1개 → 검색 → Top-5 → 리랭킹

[쿼리 확장 방식]
사용자 질문 1개 → GPT-4o-mini → 변형 쿼리 3개 추가 생성
  → 원본 + 변형 3개 = 최대 4개 쿼리로 각각 검색
  → 결과 합산 (Union, 중복 제거) → 평균 14.2개 문서 수집
  → 리랭킹으로 Top-3 선별
```
