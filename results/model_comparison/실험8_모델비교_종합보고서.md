## 3. 핵심 분석

### 3-1. 평균 품질 기준: Llama 3.3 70B 우세

Llama가 1.629로 가장 높고, GPT-4o-mini 1.581, Gemini 1.468 순.
그러나 세 모델 모두 1.5~1.6 수준으로 통계적 차이는 크지 않음.

### 3-2. 완전정답률 기준: GPT-4o-mini 우세

GPT-4o-mini는 **확실하게 알면 3점**, 모르면 0점 — **양극화 패턴**.
Llama는 **2점(대부분 정확)이 많음** — 꾸준히 부분적으로 맞춤 **안정적 패턴**.

```
GPT-4o-mini:  ████████████ 3점 22개  █████ 0점 21개  ← 양극화
Llama 3.3:    ████████ 3점 17개  ████████████ 2점 21개  █████ 0점 16개  ← 안정적
Gemini Flash: ████████████ 3점 21개  ████ 0점 22개  ← GPT와 유사
```

### 3-3. 오답 회피율 기준: Llama 3.3 70B 우세

오답(0점) 비율: Llama 25.8% < GPT-4o-mini 33.9% = Gemini 35.5%
**Llama가 "모르겠다" 대신 부분 정보라도 제공하는 경향**.

### 3-4. 응답 속도: Llama 3.3 70B 압도적 우세

Llama 5.56초 < GPT-4o-mini 8.14초 < Gemini 14.0초
**Gemini는 품질 대비 속도가 가장 느림**.

### 3-5. 카테고리별 특이점

- **diet, general**: 세 모델 모두 비슷한 성능 (GPT/Llama 동점)
- **research, review**: Llama와 Gemini가 GPT보다 높음
  → 학술 논문 스타일 질문에서 Llama/Gemini가 상대적으로 강함

## 5. 결론

> **Llama 3.3 70B (Groq 무료 API)** 가 세 모델 중 가장 균형잡힌 성능.  
> 무료임에도 유료 GPT-4o-mini보다 평균 품질이 높고, 오답률이 낮으며, 응답도 빠름.  
> **현재 챗봇을 Llama로 교체하는 것을 검토할 만함.**

단, 완전정답률(3점)만 보면 GPT-4o-mini(35.5%)가 가장 높아,  
"정확히 아는 것만 말하는" 신뢰도 측면에서는 GPT가 여전히 강점.
