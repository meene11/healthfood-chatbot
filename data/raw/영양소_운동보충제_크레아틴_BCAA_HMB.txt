== 크레아틴(Creatine) ==

정의: 체내에서 아르기닌·글리신·메티오닌으로 합성되는 물질. 근육에 크레아틴 인산 형태로 저장. 고강도 운동 시 ATP(에너지) 재합성에 사용.

효능:
고강도 운동(5~10초) 시 에너지 공급 향상 → 근력, 파워 향상
근육 단백질 합성 촉진, 근육 분해 억제
근육 세포 내 수분 저류 → 근육 부피감 증가(단기적)
회복 속도 향상

다이어트 시 역할:
체중 감량 중 근육량 유지에 도움. 근력 운동 퍼포먼스 유지 → 더 효과적인 운동 가능. 근손실 방지 → 기초대사량 보호.

복용법:
로딩 단계(선택): 첫 5~7일간 하루 20g(5g×4회) → 빠른 근육 내 저장
유지 단계: 하루 3~5g 지속
물 충분히 마실 것 (세포 내 수분 증가로 탈수 방지)

권장 용량: 시중 제품 대부분 하루 3g 내외. 연구에 따라 3~20g 다양.
복용 타이밍: 운동 전후 모두 효과적. 취침 전 복용도 괜찮음.

부작용:
초기 체중 증가 (수분 저류, 약 1~2kg) → 다이어트 중 심리적 부담 가능
신장 기능 저하자는 주의 (신장 부담 가능성 → 의사 상담 필요)
위장 불편감 (대용량 복용 시)

형태: 크레아틴 모노하이드레이트(가장 연구 많고 효과 검증), 크레아틴 에틸에스터, 크레아틴 HCL 등

== BCAA (분지쇄아미노산, Branched Chain Amino Acids) ==

정의: 필수아미노산 중 류신(Leucine), 이소류신(Isoleucine), 발린(Valine) 3종의 복합체.

효능:
류신: mTOR 경로 자극 → 단백질 합성 유도 (근육 합성의 핵심 신호)
이소류신: 글루코스 섭취 촉진, 에너지 대사
발린: 근육 에너지 공급
운동 중 근육 에너지원으로 사용 → 근육 분해(이화작용) 억제
근육통 감소, 회복 촉진
운동 중 피로 지연(세로토닌 전구체인 트립토판 경쟁 억제)

다이어트 시 역할:
식이 제한 중 근손실 방지 (칼로리 부족 상황에서 근육 보호)
운동 중간 복용으로 근육 이화 억제
공복 유산소 운동 전 복용 → 근손실 방지

주의: BCAA는 필수아미노산만 제공 → 전체 단백질 요구량 충족 불가. 완전 단백질(WPI, WPC) 병행 권장.
복용 타이밍: 운동 직전/중간/직후. 다이어트 중에는 운동 전 복용 권장.
권장 용량: 하루 5~20g. 류신:이소류신:발린 = 2:1:1 비율 제품이 일반적.

== HMB (Beta-Hydroxy Beta-Methylbutyrate) ==

정의: 필수아미노산 류신의 대사산물. 류신 섭취량의 약 5%만 HMB로 전환됨.

효능:
단백질 합성 촉진 + 단백질 분해 억제 (이중 작용)
근육 단백질 분해 억제 효과는 크레아틴보다 강하다는 연구도 있음
근육 회복 촉진
지구력 운동 퍼포먼스 향상
노인·초보자에서 근감소 예방 효과 연구 다수 (KCI 논문)

다이어트 시 역할:
칼로리 제한 + 유산소 운동 병행 시 근손실 방지에 특히 효과적
체지방 감소 + 근육량 유지 동시 목표에 적합

권장 용량: 하루 3g (1g×3회 분복 권장)
복용 타이밍: 운동 전·후 + 식간

== 다이어트 목적별 보충제 조합 요약 ==
체중 감량 + 근육 유지: 단백질(WPI) + BCAA + HMB
근력 향상 + 체중 감량: 단백질 + 크레아틴
지구력 운동 중심: 단백질 + BCAA + 오메가3
노인/초보자 근감소 예방: HMB + 크레아틴 + 단백질
